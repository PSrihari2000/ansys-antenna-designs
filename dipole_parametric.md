# Dipole Antenna (Parametric Analysis)

This file contains a dipole antenna design with parametric analysis in ANSYS HFSS.

## Key Concept

Parametric analysis studies how changing parameters (like length or frequency) affects antenna performance.

## What to observe

* Effect of length variation on resonant frequency
* Changes in return loss (S11)
* Bandwidth variation

## Why this is useful

Helps understand how antenna design parameters impact performance.

## Usage

Open the `.aedt` file in ANSYS HFSS and run the parametric sweep.
