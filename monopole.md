# Monopole Antenna

This is a monopole antenna designed using ANSYS HFSS.

## Key Concept

A monopole antenna is typically λ/4 in length and requires a ground plane.

## What to observe

* Return loss (S11)
* Radiation pattern
* Effect of ground plane

## Applications

* Communication systems
* RF transmission

## Usage

Open the `.aedt` file in ANSYS HFSS to view and simulate.
