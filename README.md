# 📡 ANSYS Antenna Designs

A collection of ANSYS HFSS antenna design files for students and beginners.

## 📂 Available Designs 
- Dipole Antenna  
- Dipole Antenna (Parametric Analysis)  
- Monopole Antenna  
- Monopole Antenna (Radius Variation)  
- Circular Microstrip Patch Antenna  
- Horn Antenna  
- Yagi-Uda Antenna  

## 🎯 Purpose
During my B.Tech, I couldn’t find ready-made ANSYS antenna files before lab exams.
So I’m sharing my designs to help students learn and save time.

## 🛠 How to Use
1. Download the `.aedt` file  
2. Open it in ANSYS HFSS  
3. Run simulation or view results
