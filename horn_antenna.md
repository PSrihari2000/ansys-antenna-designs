# Horn Antenna

This is a horn antenna designed using ANSYS HFSS.

## Key Concept

Horn antennas are high-gain antennas used to direct electromagnetic waves efficiently.

## Features

* High directivity
* Wide bandwidth
* Low loss

## What to observe

* Radiation pattern (highly directional)
* Gain
* Beamwidth

## Applications

* Radar systems
* Satellite communication
* Microwave communication

## Usage

Open the `.aedt` file in ANSYS HFSS to view and simulate.
