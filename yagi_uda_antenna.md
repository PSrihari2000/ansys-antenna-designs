# Yagi-Uda Antenna

This is a Yagi-Uda antenna designed using ANSYS HFSS.

## Key Concept

A Yagi-Uda antenna is a directional antenna consisting of a driven element, reflector, and directors.

## Features

* High gain
* Directional radiation pattern
* Improved signal strength in a specific direction

## What to observe

* Radiation pattern (directional)
* Gain improvement
* Effect of directors

## Applications

* TV reception
* Wireless communication
* RF transmission systems

## Usage

Open the `.aedt` file in ANSYS HFSS to view and simulate.
