# Monopole Antenna (Wire Radius Variation)

This antenna design analyzes the effect of wire radius on monopole antenna performance using ANSYS HFSS.

## Key Concept

Changing the wire radius affects impedance, bandwidth, and radiation characteristics.

## What to observe

* Variation in return loss (S11)
* Bandwidth changes
* Effect on resonant frequency

## Why this is useful

Helps understand how physical dimensions impact antenna performance.

## Usage

Open the `.aedt` file in ANSYS HFSS and run the parameter variation to observe results.
