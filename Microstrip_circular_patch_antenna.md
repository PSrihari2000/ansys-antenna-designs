# Circular Microstrip Patch Antenna

This is a circular microstrip patch antenna designed using ANSYS HFSS.

## Key Concept

Microstrip patch antennas are low-profile antennas widely used in wireless communication systems.

## Design Details

* Shape: Circular patch
* Typically used in GHz frequency range
* Built on dielectric substrate

## What to observe

* Return loss (S11)
* Radiation pattern
* Gain and directivity

## Applications

* WiFi
* Satellite communication
* Mobile communication

## Usage

Open the `.aedt` file in ANSYS HFSS to view and simulate.
